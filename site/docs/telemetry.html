Metric types

Every metric in a Rezolus recording is one of these three shapes. Column metadata records the type, and the viewer dispatches chart rendering accordingly.

counterUInt64, monotonic

A monotonically non-decreasing total. Rates and deltas are derived at query time via rate() / irate(). Examples: cpu_usage, syscall, blockio_operations.

gaugeInt64, instantaneous

An instantaneous value that can go up or down. Used as-is — no rate derivation. Examples: cpu_cores, memory_used, gpu_utilization.

histogramList<UInt64>, H2 bucketed

A full bucketed distribution per snapshot, not just summary percentiles. Queryable via histogram_quantiles(), histogram_mean(), histogram_sum(), etc. Examples: scheduler_runqueue_latency, syscall_latency, tcp_packet_latency.
